## O que é o Gênesis?

O **Gênesis** não é um simples gerador de templates. É uma plataforma de engenharia de software assistida por IA, projetada para atuar como um arquiteto de software sênior. Ele automatiza a criação de aplicações full-stack completas, desde a estrutura inicial do projeto até a configuração do deploy, permitindo que desenvolvedores e equipes foquem no que realmente importa: a lógica de negócio e a inovação.

A filosofia do Gênesis é baseada em três pilares:
1.  **Qualidade Extrema:** O código gerado não apenas funciona, ele é limpo, bem arquitetado, testável e segue as melhores práticas da indústria.
2.  **Desenvolvimento Acelerado:** Automatizamos as tarefas repetitivas e complexas do início de um projeto para reduzir drasticamente o tempo de lançamento no mercado.
3.  **Evolução Inteligente:** O Gênesis não abandona o desenvolvedor após a criação. Ele oferece ferramentas para evoluir e manter a aplicação de forma consistente e segura.

## Funcionalidades Principais

- **Arquétipos de Aplicação:** Gere aplicações complexas (ex: SaaS B2B, Marketplaces) com um único comando.
- **Código Full-Stack:** Frontend, backend e banco de dados, tudo gerado e conectado.
- **Testes Automatizados:** Estrutura de testes unitários e end-to-end pronta para uso.
- **CI/CD Integrado:** Workflow de Integração Contínua para GitHub Actions gerado automaticamente.
- **Copiloto de IA (Roadmap):** Assistente para adicionar novas funcionalidades, APIs e modelos de forma inteligente.
- **Arquiteto Visual (Roadmap):** Interface web para desenhar a arquitetura da sua aplicação.

## Roadmap de Desenvolvimento

O projeto será desenvolvido em três fases estratégicas:

- **✅ Fase 1: A Fundação - O Arquiteto de Software Especialista**
  - **Foco:** Geração de aplicações de alta qualidade baseadas em arquétipos.
  - **Entregável:** CLI `genesis-cli` com o comando `new`.

- **⏳ Fase 2: A Evolução - O Copiloto de IA**
  - **Foco:** Comandos na CLI para modificar e evoluir o código gerado.
  - **Entregável:** Comandos `add:*`, `write:tests`, etc.

- **▶️ Fase 3: A Abstração - O Arquiteto Visual**
  - **Foco:** Plataforma web com interface visual para design de aplicações.
  - **Entregável:** `app.genesis.com`.

## Como Começar (Em breve)

Quando a Fase 1 estiver concluída, você poderá começar com um simples comando:

```bash
npx @genesis/cli new meu-projeto-incrivel
```
